## Description
**Evil-Gpt_5** is a streamlined, Bash-based terminal interface designed to interact with the EvilGPT engine. It brings high-performance, unfiltered AI capabilities directly to your command line.

---

## Installation 
Clone the repo and cd into the directory
```bash
git clone https://github.com/Vx-main/Evil-Gpt_5.git
cd Evil-Gpt_5
```

## Fix File Permissions
Before the script can execute, you must grant it execution rights. Navigate to the folder containing your script and run:

```bash
chmod +x evilgpt
```

## Global Setup

To use `Evil-Gpt_5` as a global command from any directory, follow these steps to set up your environment.
```bash
sudo mv evilgpt /usr/local/bin/evilgpt
```

## Usage 
```bash
evilgpt "You Prompt"
```
